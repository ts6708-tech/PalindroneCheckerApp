# PalindroneCheckerApp
/ --------------------------------------------
// Project Name: PalindromeCheckerApp
// --------------------------------------------

// Running Procedure
// 1. Compile the Java file using javac
// 2. Run the program using java command
// 3. Enter a string or number
// 4. Program checks if it is palindrome
// 5. Displays the result

// --------------------------------------------
// Flow of Project
// --------------------------------------------

// Step 1: Start the program
// Step 2: Take input from user
// Step 3: Reverse the input string
// Step 4: Compare original and reversed string
// Step 5: If both are same → Palindrome
// Step 6: Else → Not a Palindrome
// Step 7: End program

// Objective
// --------------------------------------------

// • To implement basic algorithm design
